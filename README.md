DailyFix is a modern and user-friendly home service booking web application where users can browse various services, view details, read reviews, and book professional home services instantly.

## Purpose of the Website

DailyFix aims to provide a smooth and professional service-booking experience where users can:

Explore a wide range of home services
View service details with pricing, ratings, and descriptions
Register and log in securely using Firebase Authentication
Manage personal profiles


## Core Features

-->User Authentication (Firebase Auth)

Email/Password Login
Persistent login state
Protected routes

--> User Profile Management

Update name & profile picture
Display user info (email, uid, status)

--> Service Listing and Details

Explore multiple categories
View service descriptions, images, and pricing
Customer ratings & reviews (UI prepared)

--> Add Services, Update and Delete services

--> Booking System 


### Technology Stack
- **Frontend**: React, Tailwind CSS  
- **State Management**: TanStack React Query  
- **Backend/API**: Custom Express/Node server 
- **Authentication**: Firebase Authentication  
- **Payments**: Integrated payment gateway (Stripe)  
- **Charts & Visualizations**: Recharts  
- **UI Enhancements**: Swiper, Framer Motion (animations), SweetAlert2 (alerts), React Icons
