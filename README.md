DailyFix is a modern and user-friendly home service booking web application where users can browse various services, view details, read reviews, and book professional home services instantly.

## Purpose of the Website

DailyFix aims to provide a smooth and professional service-booking experience where users can:

Explore a wide range of home services
View service details with pricing, ratings, and descriptions
Register and log in securely using Firebase Authentication
Manage personal profiles
